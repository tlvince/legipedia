# legipedia

Legible Wikipedia.

## Usage

1. Browse to a Wikipedia page
2. Replace everything before the first forward slash with:

        https://legipedia.herokuapp.com

## Example

<https://en.wikipedia.org/wiki/Throffer>  
<https://legipedia.herokuapp.com/wiki/Throffer>
